# Quiz Question with Revision

## Description

This project is a simple quiz application made using HTML, CSS, and JavaScript. It allows users to answer multiple choice questions and helps them revise the questions they answered incorrectly with proper explanations.

## Features

* Multiple choice questions
* Shows score at the end
* Displays performance message
* Stores wrong answers for revision
* Clean and simple user interface

## Technologies Used

HTML
CSS
JavaScript

## How to Run

1. Open the project folder
2. Run quiz.html in any browser
3. Start the quiz

## Working

User selects answers for each question. If the answer is correct, score increases. If wrong, it is saved and shown later in the revision section with explanation.
